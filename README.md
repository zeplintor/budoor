# Budoor

Budoor est une solution SaaS d'aide à la décision agricole (monitoring de parcelles, alertes et rapports agronomiques). Ce dépôt contient la version applicative. Pour des raisons commerciales, la documentation technique détaillée, les scripts de déploiement et les guides d'intégration ont été archivés dans une branche privée.

Si vous êtes intéressé par une licence, une collaboration ou l'achat du produit, contactez-nous à contact@budoor.example (ou ouvrez une issue privée sur le repo).

Fonctionnalités clés (aperçu public)
- Cartographie interactive et gestion de parcelles
- Prévisions météorologiques et analyses de terrain
- Génération de rapports agronomiques
- Intégrations paiement et notifications (configuration réservée aux partenaires)

Pour les contributeurs techniques : nous n'exposons pas publiquement les étapes d'installation et les clés d'API dans ce dépôt.
